Square Platformer is a 2D platformer game developed independently for Android using Unity.
The game features 20 unique levels, each designed with different layouts and challenges. Players progress through levels using precise movement and jumping mechanics. The project includes in-game advertisements and rewarded ads, integrated as part of the progression and monetization system.
The game is currently in closed testing on Android.
## Key Features
- 20 handcrafted levels with increasing difficulty
- Unique level design for each stage
- Smooth player movement and jump mechanics
- In-game advertisements
- Rewarded ads integration
- Level-based progression
